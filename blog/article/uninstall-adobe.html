卸载Adobe软件 2019.7.14

  • 首页
  • 博客
  • 文章

有时候我们会因为某些原因卸载Adobe，通常情况下，通过Creative Cloud客户端和控制面板就可以卸载程序，但在某些情况下，会发生下载失败的情况。这个时候，就需要用其他方法来强制清除 。

本文，就介绍一下官方推荐的清除工具：Creative Cloud Cleaner Tool 。需要注意，此工具没有可视化界面 ，需要使用命令行工具输入字母或数字，因此使用此工具可能会用到一点点英语。

下载地址：
点此下载；
如果您不相信上面的链接，您可以点击此处跳转到Adobe官网下载。
你也可以复制下面这个链接自行下载：
https://helpx.adobe.com/creative-cloud/kb/cc-cleaner-tool-installation-problems.html

使用方法

右键单击下载的Adobe Creative Cloud Cleaner Tool.exe文件,然后选择”作为管理员运行”；或双击Adobe CreativeCloudCleanerTool.exe以运行 EXE 文件 。

按照以下顺序操作屏幕上的说明:

1、选择您的语言:键入英语的e或日语的 j,然后按Enter。

2、查看 Adobe 最终用户许可协议：键入y以接受或n表示拒绝(如果拒绝,脚本将停止)。按 Enter。

3、键入与要选择的选项对应的数字,然后按Enter。例如,如果要卸载 Creative Cloud 应用,请输入数字4。

4、从屏幕上显示的选定选项的应用程序列表中,通过键入与应用名称对应的编号,选择要删除的应用程序,然后按Enter。

5、确认要删除产品：键入 y,然后按 Enter 。

6、当您看到消息“Adobe Creative Cloud Cleaner Tool completed successfully”时,按Enter并重新启动计算机。

原文链接： https://helpx.adobe.com/creative-cloud/kb/cc-cleaner-tool-installation-problems.html
